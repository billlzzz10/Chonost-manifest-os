# Agent Instructions

This document provides instructions for AI agents working on the Chronost-manifest-os project.

## Development Workflow

1.  **Understand the Task:** Before writing any code, make sure you understand the task requirements.
2.  **Create a Plan:** Outline the steps you will take to complete the task.
3.  **Write Code:** Implement the required changes.
4.  **Test Your Changes:** Run the project's tests to ensure your changes are correct and do not break existing functionality.
5.  **Submit Your Changes:** Create a pull request with a clear title and description.

## Coding Style

- **Rust:** Follow the official Rust style guidelines.
- **TypeScript/React:** Follow the official TypeScript and React style guidelines.
- **Tailwind CSS:** Use the Tailwind CSS utility classes for styling.

## Project Structure

- `crates/`: Contains the Rust workspace crates.
- `frontend/`: Contains the frontend application.
- `docs/`: Contains project documentation.
- `justfile`: Defines tasks for automating common development tasks.
- `README.md`: Provides an overview of the project.
